# Docs / ドキュメント

## English

Working logs and references for the Push_swap pair project (So + Javier).
All entries are **bilingual (English + 日本語)** so both partners can read and contribute.

### Structure

| Folder | Purpose |
| --- | --- |
| `subject/` | Original subject PDF and Japanese translation |
| `pair_communication/` | Shared log of pair meetings, decisions, open questions |
| `work_log/` | So's daily work log |
| `learning_log/` | Concept notes from studying for the project |

### Conventions

- Dated entries: `YYYY-MM-DD_short-topic.md` (e.g., `2026-05-20_kickoff.md`)
- Topic-only entries: `topic-name.md` (kebab-case)
- Every entry has `## English` and `## 日本語` sections
- Commit messages: `docs(<area>): <summary>` — e.g., `docs(pair): add kickoff notes`

## 日本語

Push_swap のペア課題(So + Javier)用の作業ログ・参考資料です。
全エントリは **日英バイリンガル** で記述します(両者が読み・追記できるように)。

### 構成

| フォルダ | 目的 |
| --- | --- |
| `subject/` | 課題 PDF と日本語訳 |
| `pair_communication/` | ペアでの議論・決定事項・未解決事項を共有するログ |
| `work_log/` | So の日次作業ログ |
| `learning_log/` | 課題理解のために学んだ概念のメモ |

### 規約

- 日付付きエントリ: `YYYY-MM-DD_short-topic.md`(例: `2026-05-20_kickoff.md`)
- トピック単位のエントリ: `topic-name.md`(kebab-case)
- 各エントリは `## English` と `## 日本語` の両セクションを持つ
- コミットメッセージ: `docs(<area>): <summary>`(例: `docs(pair): add kickoff notes`)
