# Learning Log / 学習ログ

## English

Concept notes from studying topics needed to understand and defend the project.
One file per topic: `topic-name.md` (kebab-case, e.g., `big-o-notation.md`, `radix-sort.md`).
Use `TEMPLATE.md` as a starting point.

### Principle

**The act of restating in your own words is the learning.**
Don't paste references — restate. If you can't restate it, you haven't learned it yet, and the partner / evaluator will notice.

### Suggested topics to start with

- `big-o-notation.md` — algorithmic complexity basics
- `radix-sort.md` — what radix sort is and why it fits push_swap
- `coordinate-compression.md` — why we map values to `0..n-1`
- `linked-list-vs-array.md` — data structure trade-offs for stack operations
- `disorder-metric.md` — what disorder measures and how the formula works
