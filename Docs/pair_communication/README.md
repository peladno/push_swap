# Pair Communication Log / ペアコミュニケーションログ

## English

Shared record of conversations with the pair partner (Javier).
Each meeting (or significant async exchange) = one file.
Both partners are encouraged to read and append.

### When to add an entry

- Before & after a sync meeting (agenda + minutes)
- When a design decision is made
- When a question needing partner input surfaces
- When tasks are reassigned

### File naming

`YYYY-MM-DD_short-topic.md` — e.g., `2026-05-20_kickoff.md`, `2026-05-27_radix-design.md`.

See `TEMPLATE.md` for the entry format.

---

## 日本語

相方(Javier)とのやりとりを共有する場所です。
1 ミーティング(または重要な非同期やりとり)= 1 ファイル。両者が読み、必要なら追記してください。

### 追加するタイミング

- 同期ミーティングの前後(議題 + 議事)
- 設計判断を行ったとき
- 相方の意見が必要な未解決事項が出たとき
- タスク分担を変更したとき

### ファイル命名

`YYYY-MM-DD_short-topic.md` — 例: `2026-05-20_kickoff.md`, `2026-05-27_radix-design.md`

エントリ形式は `TEMPLATE.md` を参照してください。
